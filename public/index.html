Trees in Oakland

This is a map and data source of known public trees in Oakland.

Public Data

These are trees in the public right-of-way in Oakland, as recorded circa 2005-2006. In total there are 40,000 public trees in Oakland, this map currently features the trees that produce something you can eat. You can filter by season. Data from the Oakland Department of Public Works

About

Oakland Trees visualizes public trees in Oakland. We documented and improved the public tree data from the City of Oakland, and adding useful data to make the map visualization useful and meaningful, especially concerning edible fruit trees in Oakland.

This project began as an initiative for Code for Oakland. Our Edible Fruit Tree Data project (involving cleaning up this Trees dataset from the City of Oakland) won for Best Use of Public Data.

Things you can do

  • Interact with the map.
  • Download tree data.
  • Help improve the data.
  • Help keep tree data current. Sign on to github to edit the tree CSV file. Map will be periodically updated.
  • Help with tree care: The City of Oakland allows for tree plantings and for public tree prunings (due to lack of funding.)
  • Contribute more open datasets of public, shared and edible trees in Oakland to fill in the map.

Future Improvements

  • Adoption of Oakland Trees by tree lovers of Oakland.
  • Occasional meetups in Oakland to develop the projects and keep the map viable.
  • Put tree data in mongo database.
  • Add geolocated tree lookup. (Can't show 40,000 trees on the map at once, requires filter.)
  • Allow people to add shared trees.
  • Get more tree datasets for Oakland planted since 2005. Develop workflow to add these to the map.
  • Make map and add tree form embeddable so that more shared trees can go into a public registry.

History

Code for Oakland hackathon event

  • We added common names for 11,000+ tree species from USDA Plant Checklist (using Google refine)
  • Geocoded some tree data (but not all) based on address data.
  • We are asking the City of Oakland for information about when tree data was collected and by whom. (Oakland website does not supply this information.)
  • Considering creating a database of edible, medicinal and graftable fruit trees.
  • Researched seasonality data
  • Reached out to CUESA about making their local seasonal data machine readable. (http://cuesa.org/page/seasonality-chart-fruit-and-nuts)

Map & Data Development August-September 2012

  • We spoke with the City of Oakland and learned that the tree data was current as of 2005-2006.
  • We gave common names to nearly all trees.
  • We geolocated all of the trees (using the Yahoo Geocoder)
  • We made a machine-readable seasonality database of fruiting trees in the Bay Area
  • We added a seasonality filter to the map.
